### Getting Started

1. Install packagesCreate the `.env` file with the following content:

```sh
NODE_ENV=development
```

2. Install packagesInstall dependencies

```sh
yarn install
```

3. Start development

```sh
yarn start

# This command below won't open a new browser tab
# yarn start -no-open
```
